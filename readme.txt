#CSCI 548 project 

a web based application for searching restaurants based on dish name. Used information integration constructs to gather data from different types of data sources like websites and web exposed api using python scripts. Modeled and processed data into an integrated format using entity resolution techniques,python scripts, google refine and karma.Developed web interface using angular JS, JQuery and stored back end data into elastic search store. Used angular JS elastic search interface for querying and displaying results.
